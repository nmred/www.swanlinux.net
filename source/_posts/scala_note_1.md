\10.  take, drop, takeRight, dropRight这些字符串函数是做什么用的？和substring相比，它们的优点和缺点都有哪些？ 

```
	scala> val str : String = "abcdef"
	str: String = abcdef

	scala> str.take(2)
	res21: String = ab

	scala> str.drop(2)
	res22: String = cdef

	scala> str.takeRight(2)
	res23: String = ef

	scala> str.dropRight(2)
	res24: String = abcd
```
